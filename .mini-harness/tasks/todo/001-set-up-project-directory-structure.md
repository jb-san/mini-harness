# Set up project directory structure

## Description
Create the basic directory structure for a new project including src/, tests/, docs/, and config/ folders with appropriate subdirectories.

## Acceptance Criteria
- [ ] src/ directory created with main/ and utils/ subdirectories
- [ ] tests/ directory created with unit/ and integration/ subdirectories
- [ ] docs/ directory created with API/, guides/, and README.md
- [ ] config/ directory created with default config files
- [ ] All directories have appropriate .gitkeep files
