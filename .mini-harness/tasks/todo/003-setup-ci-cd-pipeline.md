# Setup CI/CD pipeline

## Description
Configure a CI/CD pipeline using GitHub Actions or similar service to run tests, linting, and build processes on every push and pull request.

## Acceptance Criteria
- [ ] .github/workflows directory created
- [ ] CI workflow file created with test, lint, and build jobs
- [ ] Workflow triggers configured for push and pull_request events
- [ ] Test job configured to run on all branches
- [ ] Build job configured to run on main/master branch only
- [ ] Workflow includes success/failure notifications
